→按住「Ctrl」键，再按「*」键，可把微型地图从半透明变成不透明，或是不透明变成半透明。

→按住「Shift」键，再按住鼠标右键拖曳，可以把摄像机视角的角度抬高。如要恢复原状，按住「Shift」键，再双击鼠标右键即可。 
#
→按住「Ctrl」键，再按住鼠标右键拖曳，可以把视角放大缩小。这时按住「Ctrl」键，再双击鼠标右键，即可恢复原状。另外使用鼠标滚轮也可以进行画面缩放。
#
→把游戏光标放在敌人身上时，光标会变成刀的图样， 这时代表目标可以攻击。
#
→击杀魔物获得的经验值，与魔物的杀伤力成正比。
   ※一起攻击魔物时，先攻击魔物的人，获得的经验值较多。
#
→按住「Ctrl」键攻击魔物时，会变成「锁定攻击状态」，将会持续自动攻击直到魔物死亡为止。想要解除自动锁定攻击状态时，将鼠标光标点选其它位置即可。
#
→想要拾取掉下来的物品，把光标指在物品上再「单击鼠标左键」即可，拾取到的物品会放在物品栏里。
#
→把光标移到物品身上「单击鼠标右键」时，可以查看物品的情报。
#
→玩家脚下的色带是玩家的HP值和SP值，绿色的表示HP值，蓝色的表示SP值。HP值到0时，表示玩家已死亡，为了提醒玩家HP值降低到25%以下时，HP值会显示为红色。
#
→EXP显示条已满的话(100%)，基础等级会上升。基础等级上升后会获得素质点数（Status Point），玩家可自由添加到想培养的素质上。
#
→EXP 表示杆下面的JOBEXP显示杆已满的话(100%)， 职业等级会上升，会获得技能点数，玩家可自由加给于想培养的技能。
#
→玩家原地不动时，HP和SP会缓慢恢复。
   ※坐下时恢复速度会提升为2倍，而当玩家负重超过50%以上时，将不会恢复HP值与SP值。
#
→「Weight(重量)」显示为最大负重量和当前携带的物品重量。玩家无法携带超过最大负重的物品，物品重量超过最大负重量时，将不能携带任何物品。若携带的物品重量超过最大负重量一半时，将会显示为红色，这时HP和SP不会自动恢复。当前负重超过最大负重的90%时，玩家将不能攻击和使用技能。
#
→「Zeny(金币)」是在《仙境传说》的世界里所使用的货币单位。 
   ※玩家所拥有的金币会在交易或购买物品时自动扣减。
#
→基础等级上升后，会得到素质点数（Status Point），玩家可利用素质点数培养素质。
   ※随着等级提升，升级后获得的素质点数也会提升。
#
→「STR(力量)」对玩家物理攻击力有直接的影响，力量增加时，最大负重量也将会增加，将可携带更多物品。力量增加将会增加近距离的攻击力。
#
→「DEX(灵巧)」影响玩家的命中率，也将影响到技能的咏唱的速度。
使用弓类武器进行攻击时，灵巧值将会直接影响物理攻击力。
#
→「AGI(敏捷)」和玩家的行动速度有直接的关系。
数值增加时，回避率（Flee）将会增加，闪躲敌人攻击的机率也会上升。同时，各种武器的攻击速度也将会提升。
#
→「VIT(体质)」影响玩家的防御力和HP值。
   数值提高时，防御力也将会增加，遭受敌人攻击时可以减少伤害，同时也会提高最大HP值并加快恢复HP的速度。
#
→「INT(智力)」和玩家的魔力（SP）有直接的关系，随着数值提高，魔法攻击力（MATK）和魔法防御力（MDEF）也会相应地增加，同时还会技能咏唱速度产生影响。
#
→「LUK(幸运)」玩家的幸运对各玩家种素质都会产生一定的影响，主要会影响必杀攻击率和完全回避率，此外还会细微地影响攻击力、回避率和命中率。
#
→按下「Print Screen」 or 「Scroll Lock」会对当前画面进行截图，照片会保存在「ScreenShot」文件夹内。

→「F10」可以调整对话窗口大小。

→「F12」可以开启关闭快键键菜单。  
   ※将经常使用的物品或是技能拖曳到对应的空格上，就可以通过直接按快速键进行使用。
#
→「Insert」坐下或站立 。 
※将「/坐下」或是「/站立」这两个指令在对话栏输入也可以使用。在坐下的状态时HP、SP回复速度是站立时的两倍。
#
→「Alt」+「End」可以把角色下方 「HP / SP」 图示开启/关闭。

→「Alt」+「Home」可以把可移动的地区图示开启/关闭。
#
→「Alt」+「=」切换输入状态，全角转半角。用全角输入的命令不会执行，这时候切换成半角输入即可。
#
→「Alt」+「F10」可以把聊天窗口开启/关闭。
#
→「Alt」+「M」预设置各种表情符号 / 重复的聊天的文字。
#
→按「Alt」+「1~9」可以使用「ALT」+「M」预设好的内容。
#
→ 「Ctrl」+「-,=,w」可以使用 剪刀 / 石头 /布 的表情符号。

→「/H」显示对基本的指令说明。

   ※也可以按「Alt」+「0」在窗口上执行。

   ※也可以按「Alt」+「0」在窗口上执行。
#
→「/EFFECT」可以把游戏里的各种效果图示开启/关闭。   
   ※关闭效果时，可以使游戏更加顺畅。
#
→「/SKIP」开启/关闭画面同步更新。
   ※当画面关闭时，因计算机配备不同会造成游戏速度下降，当游戏画面出现Lag时建议关闭。

#
→把光标放在想邀请的玩家身上，点选鼠标右键，会显示「邀请参加组队」选项，利用这一选项，可组成队伍。
#
→「/LEAVE」可以脱离自己加入的队伍。 
   ※脱离时不会询问 (Y/N)，所以要慎重考虑。
#
→「/EXPEL 【角色名称】」组队长可将组队员，从队伍中请离。
   ※请离时不会询问 (Y/N)，所以要慎重考虑。
